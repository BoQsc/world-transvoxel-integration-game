# World Transvoxel Terrain Addon License Scope

Files in `addons/world_transvoxel_terrain/` are 0BSD unless a file explicitly
states otherwise.

This addon is a terrain layer above `world-transvoxel`; it does not include the
`world-transvoxel` addon, Eric Lengyel's MIT Transvoxel source, or Transvoxel
lookup data.

If a game distributes this addon together with `world-transvoxel`, that
distribution must retain the license notices shipped by `world-transvoxel`.
