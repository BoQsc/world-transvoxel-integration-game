# Third-party terrain textures

The terrain albedo textures in this folder are intended as standard reference
textures for the integration game while the material/biome system is being
validated.

Source: [Poly Haven](https://polyhaven.com/)

License: [CC0 / public domain](https://polyhaven.com/license)

Poly Haven states that its HDRIs, textures, and 3D models are released under
CC0. CC0 does not require attribution, but this notice keeps the repository
auditable and makes the source of the bundled reference textures explicit.
