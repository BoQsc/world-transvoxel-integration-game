# Terrain texture sources

The terrain albedo textures in this folder are intended as standard reference
textures for the integration game while the material/biome system is being
validated.

Some files are third-party CC0 reference textures. Some files are user-supplied
generated reference textures. This notice keeps the repository auditable and
makes the source of the bundled reference textures explicit.

Poly Haven source: [Poly Haven](https://polyhaven.com/)

Poly Haven license: [CC0 / public domain](https://polyhaven.com/license)
